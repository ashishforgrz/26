JavaScript let -
The let keyword allows you to declare a variable with block scope
var x = 10;
// Here x is 10
{
  let x = 2;
  // Here x is 2
}
// Here x is 10

JavaScript const -
The const keyword allows you to declare a constant
var x = 10;
// Here x is 10
{
  const x = 2;
  // Here x is 2
}
// Here x is 10
